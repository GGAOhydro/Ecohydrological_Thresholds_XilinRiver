# Ecohydrological Threshold Analysis in the Xilin River Basin

This repository contains the custom scripts used for ecohydrological threshold analysis in the Xilin River Basin under long-term climate–grazing conditions.

The analysis framework combines process-based ecosystem simulations with interpretable machine learning approaches, including:

- Random Forest (RF) modeling
- SHAP (Shapley Additive Explanations) analysis
- ALE (Accumulated Local Effects) analysis
- Ecological threshold detection
- Bootstrap uncertainty analysis

## Repository Contents

### `Batch_dnd.ipynb`
Notebook for DNDC batch processing.

### `RF_SHAP_ALE_Bootstrap.ipynb`
Notebook for:
- Random Forest modeling
- SHAP analysis
- ALE analysis
- Ecological threshold identification
- Bootstrap uncertainty assessment

## Software Environment

The analyses were conducted using:

- Python 3.x
- Jupyter Notebook

## Data Availability

Due to the large size of the DNDC simulation outputs and restrictions associated with third-party datasets, the complete datasets used in this study are not publicly distributed.

The repository provides the analysis scripts necessary for methodological transparency and reproducibility.
